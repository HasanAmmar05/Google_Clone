# Google Clone

This project is a Google clone built with React, TypeScript, Vite, and Tailwind CSS. It utilizes an API from RapidAPI to fetch search results and includes features such as voice search and light/dark mode functionality.

## Features

- Google-like search functionality
- Voice search capability
- Light and dark mode toggle
- Responsive design using Tailwind CSS
- Fast development and building with Vite
- Images/Videos/News functioning fiteration wasn't implemented due to the API's limitations

## Technologies Used

- [React](https://reactjs.org/) with [TypeScript](https://www.typescriptlang.org/) (TSX)
- [Vite](https://vitejs.dev/) for fast development and building
- [Tailwind CSS](https://tailwindcss.com/) for styling
- [RapidAPI](https://rapidapi.com/) for search results

## Prerequisites

Before you begin, ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v14 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)


## Usage

1. Enter a search query in the search bar or click the microphone icon to use voice search.
2. Press Enter or click the search button to see the results.
3. Toggle between light and dark mode using the theme switch in the header.
